The three main diffrences between Web 1, Web 2 and Web 3:

  • Web 1 foreced the user to enter full URL address to get into website.
    Web 2 and 3 are both able the user enter only partly URL address.
  • Web 1 was not interactive with the user at all.
    Web 2 and Web 3 are both very interactive with the user compared to web 1.
  • Web 1 mostly focused on the company, while web 2 focused on the communitiy.
    At web 3 the focus is individual.
